# Mecánica Seriols

Sitio web para el taller **Mecánica Seriols**, desarrollado con [Astro](https://astro.build) y desplegado en **Cloudflare Pages**.

Repositorio: [github.com/escamilaweb/mecanicaseriols](https://github.com/escamilaweb/mecanicaseriols)

## Comandos

| Comando | Acción |
| --- | --- |
| `npm install` | Instala dependencias |
| `npm run dev` | Servidor local |
| `npm run build` | Build → `dist/` |
| `npm run preview` | Preview del build |
| `npm run pages:dev` | Preview con Pages Functions |

Ver **[DEPLOY.md](./DEPLOY.md)** para Cloudflare Pages y variables Resend.
